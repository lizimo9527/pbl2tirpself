# TripSelf - 旅行自助服务平台

## 项目简介

TripSelf 是一个一站式个性化旅游规划与社交平台，整合工具、内容与社交三大功能，为用户提供智能攻略制作、分类社区和搭子匹配服务。

## 功能特性

### 🎯 核心功能
- **智能攻略制作器**：创建个性化旅游攻略，支持地图标点、路线规划
- **发帖社区**：分享旅行经验，浏览他人攻略和帖子
- **匹配搭子**：基于旅行人格测试，智能匹配旅行伙伴
- **用户中心**：管理个人攻略、帖子和搭子信息

### 🎨 界面特色
- 现代化绿色主题设计
- 响应式布局，支持多端适配
- 直观的用户交互体验
- 丰富的图标和动画效果

## 技术栈

### 前端技术
- **框架**：uni-app (Vue.js)
- **样式**：CSS3 + Flex布局 + Grid布局
- **图标**：SVG图标 + Emoji
- **地图**：高德地图API

### 开发工具
- **IDE**：HBuilderX / VS Code
- **构建工具**：Vite
- **版本控制**：Git

## 快速开始

### 环境要求
- Node.js >= 14.0.0
- HBuilderX 或支持uni-app的IDE

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
# HBuilderX 运行到微信小程序
# 或使用命令行
npm run dev:mp-weixin
```

### 构建发布
```bash
# 构建微信小程序
npm run build:mp-weixin
```

## 页面说明

### 首页 (pages/index/index)
- 应用入口，展示热门分类和推荐内容
- 提供快速入口到主要功能模块
- 支持搜索和分类浏览

### 创建攻略 (pages/create/create)
- 创建新的旅游攻略
- 设置基本信息、分类、风格和可见性
- 支持地图标点和路线规划

### 社区页面 (pages/community/community)
- 浏览和发布旅行相关帖子
- 支持分类筛选和搜索
- 互动功能（点赞、评论、分享）

### 匹配搭子 (pages/match/match)
- 旅行人格测试
- 智能匹配推荐
- 搭子信息展示

### 用户中心 (pages/user/user)
- 个人信息管理
- 我的攻略、帖子、搭子
- 系统设置

## 数据存储

### 本地存储
使用 `uni.setStorageSync` 进行本地数据持久化：
- 用户信息
- 搜索历史
- 攻略数据
- 帖子数据

### API接口
提供模拟API接口，支持：
- 用户登录注册
- 攻略CRUD操作
- 帖子管理
- 匹配算法
